## 🏆 Highlight projects

### 🌦️ The Weather Station *(1st half)*

Okay so everyone in class built a weather station. Most had a single DHT11 sensor for temperature and humidity. Mine had... a few more things.

**Sensors & hardware:**
- 🌡️ **DHT11** — temperature & humidity (the standard one everyone had)
- 🏔️ **BMP280** — air pressure & altitude
- 💨 **MQ gas sensor** — air quality & dangerous gas detection
- 🌬️ **DC motor + fan** — wind speed measurement via analog current reading on Pico's ADC pin
- 🖥️ **OLED screen + buttons** — a full menu to cycle through each sensor reading one by one

**Software side:**
- Connected to **WiFi & MQTT** to push all data to **Home Assistant**
- Built a **web interface** to visualize the live readings
- Also did some embedded system work on **Wokwi** (simulator)

It was a lot. But it worked. Every sensor, every reading, all of it. 🙂

### 📊 Kempower EV Charging — Data Analysis *(1st half)*

Worked with a real dataset from **Kempower**, a Finnish EV charging company. Analyzed their EV charging history data — cleaning, exploring, finding patterns. First real taste of data analytics on actual industry data. BTW... never touching data analysis ever again...

### 🤖 Machine Learning — Handwritten Digit & Picture Recognition *(2nd half)*

Jumped into ML using **Jupyter Notebook** and **TensorFlow**. Built and trained a model that can recognize handwritten numbers $ pictures. The kind of thing that feels like magic until you understand the math behind it — then it feels like *hard* magic.
